# Research: Advanced Literacy Metrics

## Metrics from Research:

### Writing Fluency
1. **WPM (Words Per Minute)** - Already implemented
2. **Revision Ratio** - Backspaces/total keystrokes
3. **Burst-Pause Pattern** - Cognitive processing
4. **Dwell Time** - Time per keystroke
5. **Flight Time** - Time between keystrokes

### Reading/Mouse Behavior
1. **Fixation Duration** - Time on same position
2. **Saccade Length** - Distance between movements
3. **Return Rate** - How often returns to start
4. **Zone Transitions** - Editor/Agent/Instructions

### Text Quality
1. **Lexical Diversity** - Unique words / total
2. **Syntactic Complexity** - Sentence length variation
3. **Cohesion Score** - Pronoun/connector usage
4. **Readability Index** - Flesch-Kincaid adapted

### Engagement
1. **Interaction Density** - Events per minute
2. **Response Latency** - Time between messages
3. **Exploration Score** - Time in different zones
4. **Completion Commitment** - Progress without abandonment
